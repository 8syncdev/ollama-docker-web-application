### **Lộ trình học Django Ninja + ReactJS + HTML, CSS, JS, TS + Python Foundation (45 buổi)**  

---

### **Mục tiêu khóa học:**  
- **Python Foundation**: Hiểu và làm chủ các khái niệm lập trình Python cơ bản, tạo nền tảng vững chắc cho việc phát triển backend.  
- **Frontend Development**: Thành thạo HTML, CSS, JavaScript (JS) và TypeScript (TS), kết hợp chúng để xây dựng giao diện người dùng hiện đại.  
- **Backend Development**: Làm chủ Django Ninja để phát triển API mạnh mẽ và bảo mật.  
- **Fullstack Project**: Tích hợp ReactJS với Django Ninja để phát triển một dự án thực tế từ đầu đến cuối.  
- **Triển khai và bảo mật**: Đưa dự án lên môi trường thực tế, tối ưu hóa hiệu năng và bảo mật.  

---

### **Tổng quan lộ trình:**  
1. **Giai đoạn 1 (Buổi 1–10)**: Python Foundation.  
2. **Giai đoạn 2 (Buổi 11–20)**: HTML, CSS, JS, TS cơ bản.  
3. **Giai đoạn 3 (Buổi 21–30)**: Django Ninja cơ bản và xây dựng API Backend.  
4. **Giai đoạn 4 (Buổi 31–40)**: ReactJS cơ bản và phát triển giao diện người dùng.  
5. **Giai đoạn 5 (Buổi 41–45)**: Triển khai, bảo mật và tối ưu hóa dự án.  

---

#### **Giai đoạn 1: Python Foundation (Buổi 1–10)**  
**Buổi 1: Làm quen với Python và cài đặt môi trường**  
- Cài đặt Python, IDE (Visual Studio Code, PyCharm).  
- Viết chương trình Python đầu tiên: "Hello, World!".  
- Hiểu cơ bản về biến, kiểu dữ liệu, và toán tử.  
- Thực hành: Tính tổng, hiệu, tích, thương hai số.  

**Buổi 2: Cấu trúc điều khiển**  
- Câu lệnh điều kiện: `if`, `elif`, `else`.  
- Vòng lặp: `for`, `while`.  
- Thực hành: Tính dãy Fibonacci và kiểm tra số nguyên tố.  

**Buổi 3: Hàm và cấu trúc dữ liệu cơ bản**  
- Tạo hàm với `def`.  
- Làm việc với danh sách, tuple, dictionary, và set.  
- Thực hành: Viết chương trình quản lý danh sách sinh viên.  

**Buổi 4: File I/O và xử lý ngoại lệ**  
- Đọc và ghi file với Python.  
- Xử lý ngoại lệ bằng `try`, `except`.  
- Thực hành: Lưu trữ dữ liệu sinh viên vào file CSV.  

**Buổi 5: Lập trình hướng đối tượng (OOP)**  
- Giới thiệu lớp, đối tượng, thuộc tính và phương thức.  
- Kế thừa và đa hình.  
- Thực hành: Xây dựng lớp quản lý sách (Book).  

**Buổi 6: Module và thư viện Python**  
- Sử dụng các thư viện tích hợp như `math`, `datetime`, `os`.  
- Cách tổ chức và tái sử dụng code với module.  
- Thực hành: Xây dựng module tính diện tích hình học.  

**Buổi 7: Giới thiệu Flask và REST API cơ bản**  
- Cài đặt Flask, tạo ứng dụng API cơ bản.  
- Thực hành: Tạo API "Hello, World!".  

**Buổi 8: Làm việc với cơ sở dữ liệu SQLite**  
- Kết nối Python với SQLite.  
- Thực hành: Tạo cơ sở dữ liệu quản lý sách với CRUD.  

**Buổi 9: Sử dụng thư viện bên ngoài (pip)**  
- Tìm hiểu pip và cách quản lý môi trường với `venv`.  
- Cài đặt và sử dụng thư viện `requests` để gọi API.  

**Buổi 10: Tổng kết Python Foundation**  
- Tổng hợp kiến thức và xây dựng ứng dụng mini quản lý thư viện sách.

#### **Giai đoạn 3: Django Ninja cơ bản và xây dựng API Backend (Buổi 21–30)**  

**Buổi 21: Giới thiệu về Django Ninja và cài đặt môi trường**  
- Django và Django Ninja là gì?  
- Cài đặt Python, Django, Django Ninja, và các công cụ cần thiết.  
- Tạo dự án Django đầu tiên.  
- Viết API đơn giản: Hello World.  

**Buổi 22: Làm việc với Models và ORM**  
- Giới thiệu Django ORM và cách quản lý cơ sở dữ liệu.  
- Tạo các model đầu tiên (e.g., User, Book, Category).  
- Sử dụng `makemigrations` và `migrate`.  
- Thực hành: Tạo model quản lý sách.  

**Buổi 23: Tạo CRUD API cơ bản với Django Ninja**  
- GET, POST, PUT, DELETE APIs.  
- Quản lý đường dẫn với `path` và `query parameters`.  
- Thực hành: API CRUD cho quản lý danh mục sách.  

**Buổi 24: Quản lý xác thực và bảo mật cơ bản**  
- Giới thiệu OAuth2 và JWT (JSON Web Tokens).  
- Tạo hệ thống đăng nhập, đăng ký với Django Ninja.  
- Thực hành: Bảo mật API với JWT.  

**Buổi 25: Middleware và xử lý lỗi**  
- Xây dựng và sử dụng middleware trong Django Ninja.  
- Xử lý lỗi tuỳ chỉnh (custom exceptions).  
- Thực hành: Log hoạt động API.  

**Buổi 26: Upload và quản lý file trong Django Ninja**  
- Tạo API upload file (e.g., ảnh bìa sách).  
- Quản lý file tĩnh (static files) và thư viện `Pillow`.  
- Thực hành: API upload ảnh và hiển thị URL ảnh.  

**Buổi 27: Pagination, Filters, và Search API**  
- Phân trang dữ liệu với `paginator`.  
- Lọc và tìm kiếm dữ liệu nâng cao.  
- Thực hành: API phân trang danh sách sách.  

**Buổi 28: Testing và kiểm tra API với Postman**  
- Cách viết test case cho API.  
- Kiểm tra API với Postman.  
- Thực hành: Test API sách và người dùng.  

**Buổi 29: Tích hợp cơ sở dữ liệu nâng cao (PostgreSQL)**  
- Kết nối PostgreSQL thay vì SQLite.  
- Indexing, query optimization.  
- Thực hành: Xây dựng API tối ưu hóa hiệu suất.  

**Buổi 30: Xây dựng API cho dự án thực tế**  
- Tổng hợp các kiến thức để xây dựng API hoàn chỉnh.  
- Thực hành: API cho hệ thống quản lý thư viện.
